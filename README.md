# Xinhu Feng Academic Website

Static personal academic website for `https://xinhufeng.github.io/`.

## Preview Locally

```powershell
python -m http.server 8000 --bind 127.0.0.1
```

Then open `http://127.0.0.1:8000/`.

## Deploy

Create or open the GitHub repository named `xinhufeng.github.io`, place these files in the repository root, and push to the default branch. GitHub Pages will serve the site from `https://xinhufeng.github.io/`.
